# ドキュメント一覧

このディレクトリには、Shopify Next.js App Router テンプレートに関する詳細なドキュメントが含まれています。

## 📚 ドキュメント一覧

### 技術仕様書

- **[テンプレート仕様書](TEMPLATE_SPEC_JA.md)**  
  テンプレートの構造、認証フロー、データベース設計、Webhook 処理などの技術仕様を説明します。開発者がテンプレートの仕組みを理解するための詳細な技術情報が記載されています。

### セットアップガイド

- **[セットアップガイド](SETUP_JA.md)**  
  ローカル開発環境をセットアップするための詳細な手順を説明します。Shopify CLI、データベース、環境変数の設定方法などが記載されています。

### テンプレートの特徴

- **[テンプレートの特徴](TEMPLATE_FEATURES_JA.md)**

## 📖 読み方の推奨順序

1. **初めてこのテンプレートを使う場合:**
   - [セットアップガイド](SETUP_JA.md) を読んで環境をセットアップ
   - [テンプレート仕様書](TEMPLATE_SPEC_JA.md) で構造を理解

2. **テンプレートの仕組みを理解したい場合:**
   - [テンプレート仕様書](TEMPLATE_SPEC_JA.md) を参照

3. **公式テンプレートとの違いを知りたい場合:**
   - [テンプレートの特徴](TEMPLATE_FEATURES_JA.md) を参照
